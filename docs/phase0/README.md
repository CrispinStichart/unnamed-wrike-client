# Phase 0 Deliverables

This directory contains the documentation set that closes Phase 0 and prepares Phase 1.

Artifacts:
- `research-summary.md`: confirmed Wrike API findings, source links, and open questions
- `api-capability-matrix.md`: workflow-by-workflow support assessment for the README scope
- `mvp-scope.md`: explicit product scope decisions for Phase 1 and Phase 2
- `architecture-rules.md`: initial boundaries, dependencies, and written interface seams
- `testing-and-mocking.md`: quality rules, fake strategy, and coverage expectations
- `phase1-entry-checklist.md`: implementation checklist for the first coding phase
- `signoff.md`: final review note with resolved items, remaining questions, and the exact Phase 1 starting target

The root [`agents.md`](../../agents.md) file is the operational briefing for future AI-assisted sessions.
